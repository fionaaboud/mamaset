# Mamaset

Mamaset is an emotionally intelligent AI parenting companion (Telegram-first for hackathon MVP).

## What it does (MVP)
- Finds books by age/topic
- Suggests reputable YouTube videos with safety filters
- Finds local classes (location-based)
- Tracks school deadlines / reminders
- Gives evidence-based parenting guidance
- Captures memories and images for future “memory book” creation

## Repo contents
- `SYSTEM_PROMPT.md` — the core agent behavior + tone
- `docs/` — tool specs, API integration notes, architecture

## Safety / privacy
Do **not** commit credentials. Keep secrets in `.env` (ignored).
